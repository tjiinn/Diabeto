The Diabeto web application can be run through the deployed web-app on a browser through Streamlit OR via localhost.
**To run Diabeto via browser (recommended):**
1. Simply access the following link: https://diabeto-6ygerjyuhebkq5hf5vkmlc.streamlit.app/
2. Diabeto web app will load in your default browser
*Note: The app may take a while to load; if it does not load after a minute or so, refresh the page*

OR

**To run Diabeto via localhost:**
1. Ensure that files in the folder are downloaded
2. Open diabeto.py in your preferred Code Editor (e.g. VS Code)
3. Open the terminal in your Code Editor
4. Run the command "cd <your file path>" to the file path of the folder containing all prerequisites to run Diabeto
5. Run the command "streamlit run diabeto.py"
6. Diabeto web app will load in your default browser
*Note: Ensure ALL files in the folder are downloaded and located in the same folder, otherwise errors may be encountered*
